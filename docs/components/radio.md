# Radio 单选框

在一组选项中进行单选

## 引用

```js
import { Radio, regist } from "litos-ui";

regist(Radio);
```

## 演示

### 基础用法

使用

<ClientOnly>
<l-code-preview>
<textarea lang="html">
<l-radio></l-radio>
</textarea>
<div class="source">
<textarea lang="html">
  <l-radio></l-radio>
</textarea>
</div>
</l-code-preview>
</ClientOnly>
