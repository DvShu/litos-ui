轻量

除了依赖于小巧的 ph-utils 工具库外，没有任何其它依赖.

📦

兼容

同时兼容移动和PC，最低兼容为 ios 15.0.

🔥

逻辑可控

该组件库不同于其它组件库通过 npm 安装，该组件库提供脚手架通过脚手架的方式构建源码倒本地，便于修改.

🎨

样式可控

组件提供 part 外部可以根据不同 part 修改相应的样式；组件样式基于 CSS Variables 编写；确保样式可控.

🌍

类型安全

组件基于 typescript 编写，提供完整的类型.

🌈

全平台兼容

由于组件是基于 Web Component 编写的组件，所以该组件全平台兼容.
