SelectOri 原生下拉框 ​

这个是原生的下拉选择，只是对于 select 标签进行了一定的优化，通常用于单选时使用。

引用 ​

js
import { SelectOri, regist } from "litos-ui";

regist(SelectOri);
1
2
3

演示 ​

基础用法 ​

通过 options 属性设置选项或者通过 slot-default 来手动渲染选项。

API ​

SelectOri Attibutes ​

参数说明类型默认值
value选中项绑定值string-
name原生的 name 属性string-
disabled是否禁用booleanfalse

SelectOri Slots ​

名称说明
default选项列表

SelectOri Events ​

事件名说明回调参数
chnage选中项改变时触发(event: Event)

SelectOri Methods ​

方法名说明类型
setOptions重新设置新的选项列表(options: SelectOption[]): void
appendOptions为选项列表追加新的选项(options: SelectOption[]): void

SelectOption Attributes ​

参数说明类型默认值
value选项值string-
label选项标签string-
